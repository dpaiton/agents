# General Evaluation Rubric

This rubric is used for evaluating non-code outputs (documentation, plans, analysis, etc.). Each criterion is scored 0-2. Maximum total: **10**.

## Criteria

### Accuracy (0-2)
Is the content factually correct?

- **0** — Contains significant errors or misinformation
- **1** — Mostly accurate with minor errors or omissions
- **2** — Factually correct and precise throughout

### Completeness (0-2)
Does the output address everything requested?

- **0** — Major requirements or sections missing
- **1** — Core content present but gaps exist
- **2** — All requested elements addressed fully

### Clarity (0-2)
Is the output easy to understand?

- **0** — Confusing, poorly organized, ambiguous
- **1** — Understandable but could be clearer in places
- **2** — Clear, well-organized, unambiguous

### Usefulness (0-2)
Does the output serve its intended purpose?

- **0** — Not useful for the intended audience or purpose
- **1** — Somewhat useful but requires additional work to be actionable
- **2** — Directly useful and actionable as-is

### Evidence (0-2)
Are claims supported by reasoning, examples, or references?

- **0** — Unsupported assertions with no reasoning
- **1** — Some claims supported but gaps in evidence
- **2** — All significant claims backed by reasoning, examples, or references

## Scoring Instructions

1. Read the task requirements first
2. Write reasoning before each score
3. Assign discrete integers only (0, 1, or 2)
4. Cite specific content for every score
5. Sum for total out of 10
